# Cloud Storage (Fullstack Project)

## О проекте

Это fullstack-приложение облачного хранения файлов с поддержкой загрузки, восстановления прерванных загрузок и асинхронной обработки.

---

## Основные возможности

- Регистрация и авторизация пользователей
- Загрузка файлов
- Восстановление прерванной загрузки (resumable upload)
- Ограничение дискового пространства для пользователей
- Просмотр и управление файлами и папками
- Архивирование файлов и папок
- Асинхронная обработка задач через очереди
- Очистка системы через cron-задачи

---

## Технологии

### Backend

- PHP (собственная архитектура)
- MySQL
- Очереди для асинхронной обработки (Redis)
- Docker Compose

### Frontend

- React
- TypeScript

### Инфраструктура

- Nginx для раздачи файлов
- Docker

---

## Архитектура

В проекте реализованы собственные базовые компоненты:

- DI-контейнер
- Роутер с поддержкой динамических параметров
- Простейший слой работы с БД (репозитории)
- Система очередей

Проект разбит на слои:

- HTTP (контроллеры)
- Бизнес-логика
- Работа с данными

---

## Интересные технические решения

- Реализация возобновляемой загрузки файлов на фронтенде через state machine
- Использование очередей для обработки тяжёлых задач
- Раздача файлов через Nginx для снижения нагрузки на backend
- Генерация архивов на лету
- Работа с дедлоками и утечками соединений

---

## О проекте

Этот проект был создан в учебных целях для изучения:

- архитектуры backend-приложений
